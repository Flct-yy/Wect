引导段落

使用.lead类使段落突出显示

这是一个引导段落。它比普通段落更突出，通常用于文章的开头或重要内容提示，吸引读者的注意力。

这是一个普通段落，用于对比引导段落的效果。普通段落通常用于正文内容，样式相对简洁。

<p class="lead">
  这是一个引导段落。它比普通段落更突出...
</p>
<p>这是一个普通段落...</p>

内联文本元素

Bootstrap为常见内联HTML标签提供样式

你可以使用高亮标记来突出文本

被删除的文本表示不再相关的内容

无用文本表示不再准确的内容

插入的文本表示添加到文档的内容

下划线文本作为一种替代样式

小号文本用于免责声明和细则

粗体文本用于表示重要内容

斜体文本用于强调或语气变化

<p>你可以使用<mark>高亮标记</mark>来突出文本</p>
<p><del>被删除的文本</del>表示不再相关的内容</p>
<p><s>无用文本</s>表示不再准确的内容</p>
<p><ins>插入的文本</ins>表示添加到文档的内容</p>
<p><u>下划线文本</u>作为一种替代样式</p>
<p><small>小号文本</small>用于免责声明和细则</p>
<p><strong>粗体文本</strong>用于表示重要内容</p>
<p><em>斜体文本</em>用于强调或语气变化</p>

引用

使用blockquote元素呈现引用内容

<figure>
  <blockquote class="blockquote">
    <p>设计不仅仅是外观和感觉，设计是如何运作。</p>
  </blockquote>
  <figcaption class="blockquote-footer">
    Steve Jobs <cite title="来源标题">苹果公司联合创始人</cite>
  </figcaption>
</figure>
