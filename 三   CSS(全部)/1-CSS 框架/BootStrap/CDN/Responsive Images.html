Bootstrap响应式图片

探索Bootstrap中响应式图片的各种用法和效果

1. 基础响应式图片 (.img-fluid)

使用.img-fluid类使图片具有响应式特性，它会随父容器自动缩放。

<img src="image.jpg" class="img-fluid rounded" alt="响应式图片示例">

2. 缩略图 (.img-thumbnail)

使用.img-thumbnail类为图片添加边框、内边距和圆角，使其呈现相片缩略图效果。

<img src="image.jpg" class="img-thumbnail" alt="缩略图示例">

3. 图片形状

Bootstrap提供多种形状类来改变图片的外观。

圆角 (.rounded)

圆形 (.rounded-circle)

缩略图 (.img-thumbnail)

<img src="image.jpg" class="img-fluid rounded" alt="圆角图片">
<img src="image.jpg" class="img-fluid rounded-circle" alt="圆形图片">

4. 图片对齐

使用浮动类或文本对齐类来控制图片的位置。

使用.float-start类使图片左浮动，文本环绕图片。这是报纸和杂志中常见的排版方式，在网页设计中同样有效。

Bootstrap的响应式图片功能确保图片在不同设备上都能正确显示，同时浮动效果也会根据屏幕尺寸自动调整。

使用.float-end类使图片右浮动，文本环绕图片。这种布局可以创建视觉上的平衡感，使页面设计更加动态和有趣。

结合Bootstrap的栅格系统，您可以创建出在各种屏幕尺寸上都能良好工作的复杂布局。

<img src="image.jpg" class="img-fluid rounded float-start" alt="左浮动图片">
<img src="image.jpg" class="img-fluid rounded float-end" alt="右浮动图片">
