# 严格模式
严格模式在JavaScript中启用受限制的变体，主要变化包括：

* 消除JavaScript的一些静默错误，通过显式抛出错误
* 修复了导致JavaScript引擎难以优化的缺陷
* 禁止使用未来ECMAScript版本中可能定义的语法
* 提高代码安全性，防止意外创建全局变量

### 启用严格模式
在整个脚本中启用严格模式 "use strict";   

在函数内部声明是局部作用域 (只在函数内使用严格模式):  
在函数中启用严格模式   
```
function strictFunction() {
   "use strict"; // 严格模式代码 
   // 函数内部代码 
}
```

**重要提示：**
严格模式声明必须在脚本或函数的开头才能生效！

### 迁移到严格模式
迁移步骤：
* 在脚本或函数开头添加"use strict"
* 修复因未声明变量导致的引用错误
* 移除重复的函数参数名
* 避免删除变量、函数或不可删除属性
* 修改依赖非严格模式this行为的代码
* 使用ESLint等工具识别问题
