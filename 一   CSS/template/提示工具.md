# 基础Tooltip实现
```
html
<div class="tooltip">鼠标悬停在我上面
  <span class="tooltiptext">这是提示文本</span>
</div>

css
.tooltip {
  position: relative;
  display: inline-block;
  border-bottom: 1px dotted black; /* 可选的视觉提示 */
  cursor: pointer;
}

.tooltip .tooltiptext {
  visibility: hidden;
  width: 120px;
  background-color: #555;
  color: #fff;
  text-align: center;
  border-radius: 6px;
  padding: 5px;
  position: absolute;
  z-index: 1;
  bottom: 125%; /* 显示在元素上方 */
  left: 50%;
  margin-left: -60px; /* 使用负边距实现居中 */
  opacity: 0;
  transition: opacity 0.3s;
}

.tooltip .tooltiptext::after {
  content: "";
  position: absolute;
  top: 100%; /* 在提示框下方 */
  left: 50%;
  margin-left: -5px;
  border-width: 5px;
  border-style: solid;
  border-color: #555 transparent transparent transparent;
}

.tooltip:hover .tooltiptext {
  visibility: visible;
  opacity: 1;
}
```

# 不同定位的Tooltip
```
上方显示
.tooltip .tooltiptext.top {
  bottom: 125%;
  left: 50%;
  transform: translateX(-50%);
}
右侧显示
.tooltip .tooltiptext.right {
  top: -5px;
  left: 105%;
}
下方显示
.tooltip .tooltiptext.bottom {
  top: 135%;
  left: 50%;
  transform: translateX(-50%);
}
左侧显示
.tooltip .tooltiptext.left {
  top: -5px;
  right: 105%;
}
```
# 带箭头的Tooltip
```
.tooltip .tooltiptext::after {
  content: " ";
  position: absolute;
  border-width: 5px;
  border-style: solid;
}

/* 上方箭头 */
.tooltip .tooltiptext.top::after {
  top: 100%;
  left: 50%;
  margin-left: -5px;
  border-color: #555 transparent transparent transparent;
}

/* 右侧箭头 */
.tooltip .tooltiptext.right::after {
  top: 50%;
  right: 100%;
  margin-top: -5px;
  border-color: transparent #555 transparent transparent;
}

/* 下方箭头 */
.tooltip .tooltiptext.bottom::after {
  bottom: 100%;
  left: 50%;
  margin-left: -5px;
  border-color: transparent transparent #555 transparent;
}

/* 左侧箭头 */
.tooltip .tooltiptext.left::after {
  top: 50%;
  left: 100%;
  margin-top: -5px;
  border-color: transparent transparent transparent #555;
}
```
# 淡入动画效果
```
.tooltip .tooltiptext {
  opacity: 0;
  transition: opacity 0.3s;
}

.tooltip:hover .tooltiptext {
  opacity: 1;
}
```
# 延迟显示Tooltip
```
.tooltip .tooltiptext {
  transition-delay: 0.5s;
}
```
