# 基础导航栏

``` 
html
<nav class="navbar">
  <ul class="nav-list">
    <li><a href="#">首页</a></li>
    <li><a href="#">产品</a></li>
    <li><a href="#">服务</a></li>
  </ul>
</nav>

css
.navbar {
  background-color: #333; /* 导航栏背景色 */
}

.nav-list {
  list-style-type: none; /* 移除列表默认样式 */
  margin: 0;
  padding: 0;
  overflow: hidden; /* 清除浮动影响 */
}

.nav-list li {
  float: left; /* 使列表项横向排列 */
}

.nav-list li a {
  display: block; /* 让链接充满整个li区域 */
  color: white;
  text-align: center;
  padding: 14px 16px; /* 内边距增加点击区域 */
  text-decoration: none; /* 移除下划线 */
}

.nav-list li a:hover {
  background-color: #111; /* 悬停效果 */
}
```

# Flexbox导航栏（现代推荐）
```
.nav-list {
  display: flex; /* 启用flex布局 */
  justify-content: space-around; /* 均匀分布导航项 */
  list-style: none;
  padding: 0;
  background: #333;
}

.nav-list li a {
  color: white;
  padding: 15px 20px;
  text-decoration: none;
  display: block;
  transition: background 0.3s; /* 添加过渡效果 */
}

.nav-list li a:hover {
  background: #555;
}
```
# 垂直导航栏
```
.nav-list {
  width: 200px; /* 固定宽度 */
  background-color: #f1f1f1;
}

.nav-list li a {
  display: block; /* 重要：使链接垂直排列 */
  color: #000;
  padding: 8px 16px;
  text-decoration: none;
}

.nav-list li a:hover {
  background-color: #555;
  color: white;
}
```
# 粘性导航栏
```
.navbar {
  position: sticky; /* 关键属性 */
  top: 0; /* 距离顶部0时固定 */
  z-index: 100; /* 确保导航栏在最上层 */
}
```
# 下拉菜单导航
```
html
<li class="dropdown">
  <a href="#">产品</a>
  <ul class="dropdown-menu">
    <li><a href="#">产品1</a></li>
    <li><a href="#">产品2</a></li>
  </ul>
</li>

css
.dropdown {
  position: relative; /* 为下拉菜单定位提供参考 */
}

.dropdown-menu {
  display: none; /* 默认隐藏 */
  position: absolute; /* 绝对定位 */
  top: 100%; /* 定位在父元素下方 */
  left: 0;
  background-color: #f9f9f9;
  min-width: 160px;
  box-shadow: 0px 8px 16px rgba(0,0,0,0.2);
}

.dropdown:hover .dropdown-menu {
  display: block; /* 鼠标悬停时显示 */
}
```
