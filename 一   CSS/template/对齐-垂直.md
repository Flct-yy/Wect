# 垂直对齐

## 单行文本垂直居中
```
.single-line {
  height: 100px;
  line-height: 100px; /* 等于容器高度 */
}
```

## 多行文本/元素垂直居中

#### Flexbox (推荐)

```
.parent {
  display: flex;
  align-items: center; /* 垂直居中 */
}
```

#### 绝对定位(absolute) + transform

```
.child {
  position: absolute;
  top: 50%;
  transform: translateY(-50%);
}
```

#### Grid

```
.parent {
  display: grid;
  align-items: center;
}
```

#### 表格布局
css
.parent {
  display: table;
}
.child {
  display: table-cell;
  vertical-align: middle;
}
