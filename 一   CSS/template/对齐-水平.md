# 水平对齐

## 文本水平对齐
```
左对齐（默认）
.text-left { text-align: left; }
居中
.text-center { text-align: center; }
右对齐
.text-right { text-align: right; }
```

## 块级元素水平对齐


#### margin auto
```
.center-block {
  width: 200px;
  margin-left: auto;
  margin-right: auto;
}
```

#### Flexbox
```
.parent {
  display: flex;
  justify-content: center; 
  /* 可选：flex-start | center | flex-end | space-between | space-around */
}
```

#### Grid
```
.parent {
  display: grid;
  place-items: center; /* 同时水平和垂直居中 */
}
```
